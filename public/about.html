Absolute color management

With Coleure you can start with a great selection (inspired by PAWNTONE®) for your next project. The best part, however, is that you can mix, compare, test and even edit colors.

Keyboard shortcuts

Here’s a reward since you made it this far:

  • Hold alt and click on a second color to immediately compare two colors—this also works while mixing. We don’t really know how to describe this, but if you also hold shift something different will happen.
  • If you haven’t figured this one out yet, you can copy the code of whatever color you’re hovering with ⌘ + C or Ctrl. + C.
